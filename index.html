Californication — Экшн от первого лица


    Задачи проекта:
  • Настройка NavMesh
  • Разработка ИИ
  • Реализация стрельбы посредством Rigidbody
  • Работа с Post-Processing Stack и материалами

Цель ботов — искать и атаковать игрока.
Часть из них патрулирует территорию по заданным точкам. Остальные случайно выбирают точку на навмеше и двигаются к ней.
Если здоровье бота падает ниже определенного значения, он пытается достичь восстанавливающей камеры.



Зеленые боты атакуют только игрока, Красные боты атакуют всех.
Если игрок отсутствует, все боты атакуют друг друга.









Четырехчасовые проекты

Эти небольшие проекты — своеобразное испытание / упражнение.
На каждый ушло менее четырех часов.

Lucky Strike — Пазл для мобильных платформ

Цель игры — сопоставить по цвету как можно больше разноцветных фигур.
Новые ряды фигур появляются через некоторое время. Ряды генерируются случайным образом.
Если игрок совершил ошибку, новый ряд появляется мгновенно. Когда любая фигура достигает линии, игра заканчивается.


One Step Closer — Раннер для VR шлема Fibrum

В разработке использован Fibrum SDK для Unity.
Цель игры — пробежать наибольшее расстояние, уворачиваясь от препятствий.
Допустимы две ошибки.

Препятствия возникают в случайных точках участка.
Уровень генерируется по мере продвижения игрока вперед. Уже пройденный участок уровня, очищенный от препятствий, переносится.
Таким образом память не заполняется мусором.





Platform Royale — Командный проект

2D платформер с элементами батлрояля.
В этом проекте я выполняю роли программиста и дизайнера уровней.
Можно запустить демо Platform Royale

Особенности проекта

Был реализован поиск путей для ИИ в 2D пространстве с видом сбоку.
NavMesh работает только в 3D проектах. В ассет сторе имеются решения только для топ-даун 2D проектов.
За основу был взят алгоритм поиска пути из проекта A-Star-Pathfinding-for-Platformers

Данный алгоритм работает с битовой картой уровня. Уровень должен быть логически разбит на свободные блоки и блоки препятствий.
Соответственно был реализован автоматический способ создания такой карты для уже существующих уровней.
Путь был получен, теперь его нужно было пройти. Для этого был реализован свой контроллер перемещения бота.


Стрельба реализована посредством рейкастов, но с учетом влияния гравитации.
Траектория выстрела всегда известна. Снаряды реалистично рикошетят от препятствий.



Этот же принцип использован для предсказания траектории полета персонажа, запущенного бустер-платформой.
Благодаря этому их легко расставлять на уровне и создавать интересные комбинации.


Безопасная зона со временем сужается.


Персонаж может отталкиваться от стен и выполнять двойной прыжок.





Немного кода, программист же

Маленький инструмент для переименовывания множества объектов в Unity. Требующие переименовывания объекты нужно сложить в корневой объект. Корневой объект затем помещается в слот окошка.

using UnityEngine;
using UnityEditor;

public class RenameObjectsWindow : EditorWindow
{
    public GameObject SourceGameObject;
    private Transform[] _objects;
    private bool _withIndex;
    private string _objectName = "New Object Name";

    [MenuItem("Examples/Rename Objects")]
    public static void ShowWindow()
    {
        RenameObjectsWindow window = (RenameObjectsWindow)GetWindow(typeof(RenameObjectsWindow), false, "Rename");
        window.position = new Rect(200, 400, 320, 100);
        window.Show();
    }

    void OnGUI()
    {
        EditorGUILayout.Space();

        SourceGameObject = EditorGUILayout.ObjectField(
            "Объект источник", SourceGameObject,
            typeof(GameObject), true) as GameObject;

        _objectName = EditorGUILayout.TextField("Имя объекта", _objectName);

        _withIndex = EditorGUILayout.Toggle("С индексом", _withIndex);
        EditorGUILayout.Space();

        if (GUILayout.Button("Переименовать"))
        {
            if (SourceGameObject)
            {
                _objects = SourceGameObject.GetComponentsInChildren();
                int index = 0;
                foreach (var obj in _objects)
                {
                    if (_withIndex)
                    {
                        obj.name = _objectName + " " + index++;
                    }
                    else
                    {
                        obj.name = _objectName;
                    }
                }
            }
        }
    }
}
