Цель игры — правильно сопоставлять фигуры по цвету.
Со временем появляются новые ряды, сгенерированные случайным образом.
Если игрок совершил ошибку, новый ряд появляется мгновенно. Когда одна из фигур достигает верхней границы поля, игра заканчивается.


One Step Closer — Раннер для VR шлема Fibrum.

В разработке использован Fibrum SDK для Unity.
Цель игры — пробежать наибольшее расстояние, уворачиваясь от препятствий.

Препятствия возникают в случайных точках участка.
Уровень генерируется по мере продвижения игрока. Уже пройденный участок уровня, очищенный от препятствий, переносится вперед.
Таким образом память не заполняется лишними объектами.

Особенности проекта

Был реализован поиск пути для ботов в 2D пространстве с видом сбоку, имеющем гравитацию.
NavMesh работает только в 3D проектах. В ассет сторе имеются решения только для 2D проектов с видом сверху.
За основу был взят алгоритм поиска пути из проекта A-Star-Pathfinding-for-Platformers

Данный алгоритм работает с битовой картой уровня. Уровень должен быть логически разбит на свободные блоки и блоки препятствий.
Был реализован автоматический способ создания битовой карты для уже существующих уровней.
Но получение пути это только половина задачи, его еще нужно пройти. Для этого был создан контроллер перемещения бота, учитывающий особенности строения уровней.


Персонаж может отталкиваться от стен и выполнять двойной прыжок.


Безопасная зона со временем сужается.


Стрельба реализована посредством рейкастов, с учетом влияния гравитации.
Траектория выстрела всегда известна. Снаряды реалистично рикошетят от препятствий.

Расширение редактора Unity

Мне нравится, когда иерархия проекта упорядочена и понятна.
Одна из вещей, которые нарушают порядок — это множество безымянных однотипных GameObject-ов.
Устав переименовывать каждый объект по отдельности, я добавил в редактор окно для переименования группы объектов за раз.
Требующие переименования объекты нужно сложить в корневой объект. Затем корневой объект нужно поместить в слот окошка.



using UnityEngine;
using UnityEditor;

public class RenameObjectsWindow : EditorWindow
{
    public GameObject SourceGameObject;
    private Transform[] _objects;
    private bool _withIndex;
    private string _objectName = "New Object Name";

    [MenuItem("Tools/Rename Objects")]
    public static void ShowWindow()
    {
        RenameObjectsWindow window = (RenameObjectsWindow)GetWindow(typeof(RenameObjectsWindow), false, "Rename");
        window.position = new Rect(200, 400, 320, 100);
        window.Show();
    }

    void OnGUI()
    {
        EditorGUILayout.Space();

        SourceGameObject = EditorGUILayout.ObjectField(
            "Объект источник", SourceGameObject,
            typeof(GameObject), true) as GameObject;

        _objectName = EditorGUILayout.TextField("Имя объекта", _objectName);

        _withIndex = EditorGUILayout.Toggle("С индексом", _withIndex);
        EditorGUILayout.Space();

        if (GUILayout.Button("Переименовать"))
        {
            if (SourceGameObject)
            {
                _objects = SourceGameObject.GetComponentsInChildren();
                int index = 0;
                foreach (var obj in _objects)
                {
                    if (_withIndex)
                    {
                        obj.name = _objectName + " " + index++;
                    }
                    else
                    {
                        obj.name = _objectName;
                    }
                }
            }
        }
    }
}
