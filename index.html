Californication — Экшн от первого лица


    Задачи проекта:
  • Настройка NavMesh
  • Разработка ИИ
  • Реализация стрельбы посредством Rigidbody
  • Работа с Post-Processing Stack и материалами

Цель ботов — искать и атаковать игрока.
Часть из них патрулирует территорию по заданным точкам. Остальные случайно выбирают точку на навмеше и двигаются к ней.
Если здоровье бота падает ниже определенного значения, он пытается достичь восстанавливающей камеры.




Зеленые боты атакуют только игрока, Красные боты атакуют всех. Если игрок отсутствует, все боты атакуют друг друга.









Четырехчасовые проекты

Эти небольшие проекты — своеобразное испытание / упражнение.
На каждый ушло менее четырех часов.

Lucky Strike — Пазл для мобильных платформ

Цель игры — сопоставить по цвету как можно больше разноцветных фигур.
Новые ряды фигур появляются через некоторое время. Ряды генерируются случайным образом.
Если игрок совершил ошибку, новый ряд появляется мгновенно. Когда любая фигура достигает линии, игра заканчивается.


One Step Closer — Раннер для VR шлема Fibrum

В разработке использован Fibrum SDK для Unity.
Цель игры — пробежать наибольшее расстояние, уворачиваясь от препятствий.
Допустимы две ошибки.

Препятствия возникают в случайных точках участка.
Уровень генерируется по мере продвижения игрока вперед. Уже пройденный участок уровня, очищенный от препятствий, переносится.
Таким образом память не заполняется мусором.
